# BancoPopular

### Descripción:

Aplicación desarrollada con JavaScript y Node js, como entorno de ejecución, para la gestión de cuentas, clientes y empleados del Banco Pupular. Proyecto realizado con el propósito de poner poner en práctica los conceptos de la programación orentada a objetos (POO).

### Conceptos aplicados:

. Composición de Clases
. Herencia
. Poliformismo
. Clases Privadas
. Clases Abstractas
. Métodos Privados
. Métodos Estaticos
. Métodos Abastractos
. Interfaces
. Duck Type
